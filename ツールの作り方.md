# 自分専用の診断ツールの作り方（初心者向けガイド）

プログラミングの知識がゼロでも、AIの力を借りれば自分だけのオリジナル診断ツールは数分で作れます。このガイドでは、その最短ルートを解説します。

---

## 1. AIへの伝え方

まず、Claude 3.5 SonnetやChatGPTなどのAIに、以下をコピーして伝えてみてください。

```text
ココナラの診断ツールを作りたいです。HTML1ファイルで完結させてください。
要件は、8つの質問に答えると、自分にぴったりの出品カテゴリを3つ提案してくれるものです。
デザインは清潔感のあるプロフェッショナルな雰囲気にして、レスポンシブ対応してください。
```

---

## 2. AIが作りやすくなる「要件定義」のコツ

AIに以下の情報を追加で伝えると、より理想に近いツールが完成します。

- **質問の内容**: 何を基準にカテゴリを選びたいか
- **判定タイプ**: いくつのタイプ（例：8タイプ）に分類したいか
- **デザインの好み**: 好きな色や、参考にしたいサイトの雰囲気
- **必要な機能**: 「最後にXでのシェアボタンが欲しい」など

---

## 3. HTMLを公開する方法

AIが生成したコードを保存しただけでは、まだあなたのパソコンの中でしか見られません。インターネットを通じて世界中に公開するために、**ホスティング（置き場所）**を用意します。

今回は、無料で使えて最も信頼できる **「GitHub Pages」** を使った方法を紹介します。

---

## 4. GitHubページで無料公開する手順

### Step 4-1: GitHubにログインする
[GitHub公式サイト](https://github.com/)にアクセスしてログイン、または新規登録をします。

### Step 4-2: 新しいリポジトリ（箱）を作成する
1. 画面右上の「**+**」ボタンをクリック ＞ 「**New repository**」を選択。
2. **Repository name**: サイトの名前を英数字で入力します（例：`my-diagnosis-tool`）。
3. **Public / Private**: 必ず「**Public**」を選択してください（無料で公開するために必要です）。
4. 「**Create repository**」ボタンをクリックします。

### Step 4-3: ファイルをアップロードする
1. 作成後の画面で「**uploading an existing file**」というリンクをクリック。
2. AIが作成した `index.html` ファイルをドラッグ＆ドロップで追加します。
3. 画面下の「**Commit changes**」ボタンをクリックして保存します。

### Step 4-4: GitHub Pagesを有効化する
1. リポジトリ上部のメニューから「**Settings**」をクリック。
2. 左サイドバーの「**Pages**」をクリック。
3. **Build and deployment** セクションの「**Branch**」で「**main**」を選択、フォルダを「**/(root)**」にして「**Save**」をクリック。

✅ **数分待つと、画面上部に「Your site is live at...」と公開URLが表示されます！**
URLは `https://ユーザー名.github.io/リポジトリ名/` という形式になります。

---

## 5. GitHubページで「できること・できないこと」

| 項目 | GitHub Pagesでできること | GitHub Pagesでできないこと |
| :--- | :--- | :--- |
| **コンテンツ** | 診断ツール、ポートフォリオ、紹介サイト | オンラインショップ、会員制サイト |
| **データ保存** | 保存機能なし（画面表示のみ） | 診断者データのデータベース保存 |
| **コスト** | **完全無料**（容量内） | サーバー費用などは一切不要 |
| **セキュリティ** | HTTPS（鍵マーク）が標準で付く | サーバー側の個人情報処理 |

---

## まとめ

まずはAIに「作って」と話しかけるところから始めてみましょう。コードを1行も書かなくても、素晴らしいツールを世界中に公開することができます！
